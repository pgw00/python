📂 과제

빠른 실행

- [E01_1.py](./E01_1.py) — 이미지 불러오기 및 그레이스케일 변환
- [E01_2.py](./E01_2.py) — 페인팅 (붓 크기 조절)
- [E01_3.py](./E01_3.py) — ROI 선택 및 추출

문제가 발생하면 (VS Code에서 링크 클릭 시 "Error loading page"):

1. 파일 존재 확인: `Chapter_01` 폴더에 `E01_1.py`/`E01_2.py`/`E01_3.py`가 있는지 확인하세요.
2. 에디터 재시작: Ctrl+Shift+P → `Developer: Reload Window`로 윈도우를 재시작하세요.
3. 직접 열기: 탐색기에서 파일을 더블클릭하거나 Ctrl+P로 파일명(`E01_1.py`)을 입력해 엽니다.
4. 개발자 콘솔 확인: Help → Toggle Developer Tools → Console에 표시된 에러로 원인 파악.
5. 링크 포맷 문제일 경우 `./E01_1.py`와 같이 상대경로로 업데이트했으니 다시 시도하세요.


1) 이미지 불러오기 및 그레이스케일 변환
	- 실제 파일: `E01_1.py`
	- 요약: 이미지를 로드하고 컬러/그레이스케일을 나란히 보여주는 기본 실습

2) 페인팅 (붓 크기 조절)
	- 실제 파일: `E01_2.py`
	- 요약: 마우스 드래그로 그림을 그리는 인터랙티브 페인터. 좌/우클릭으로 색상 분기, `+`/`-`로 붓 크기 조절, `q`로 종료 및 저장

3) ROI(관심영역) 선택 및 추출
	- 실제 파일: `E01_3.py`
	- 요약: 마우스 드래그로 사각형 영역 선택 후 ROI 추출 및 저장
