# OpenCV 실습 1주차 과제

컴퓨터비전 수업에서 OpenCV 라이브러리를 활용하여 이미지 처리와 마우스 이벤트를 이용한 인터랙티브 프로그램을 구현하였다.
각 실습은 이미지 처리의 기본 개념과 사용자 입력을 활용한 기능 구현을 목표로 한다.

1️⃣ 이미지 불러오기 및 그레이스케일 변환

설명

OpenCV를 사용하여 이미지를 불러오고 컬러 이미지와 그레이스케일 이미지를 동시에 화면에 출력하는 프로그램이다.
이미지 처리의 가장 기본적인 과정인 이미지 로딩과 색상 변환을 실습한다.

주요 기능

- `cv.imread()`를 사용하여 이미지 파일을 불러오기

- `cv.cvtColor()`를 이용해 컬러 이미지를 그레이스케일로 변환

- `numpy.hstack()`을 사용하여 원본 이미지와 그레이스케일 이미지를 가로로 결합

- `cv.imshow()`로 결과 화면 출력

- `cv.waitKey()`를 통해 키 입력 시 프로그램 종료

사용 기술

- OpenCV
- NumPy
- Python

2️⃣ 페인팅 프로그램 (붓 크기 조절 기능)

설명

마우스 입력을 이용해 이미지 위에 그림을 그릴 수 있는 간단한 페인팅 프로그램을 구현하였다.
키보드 입력을 통해 붓의 크기를 조절할 수 있으며, 마우스 버튼에 따라 다른 색으로 그림을 그릴 수 있다.

주요 기능

- 마우스 드래그를 이용한 그림 그리기

- 좌클릭 : 파란색 브러시

- 우클릭 : 빨간색 브러시

- + 키 입력 시 붓 크기 증가

- - 키 입력 시 붓 크기 감소

- 붓 크기 범위 : 최소 1 ~ 최대 15

- q 키 입력 시 프로그램 종료

사용 기술

- OpenCV 마우스 이벤트 (`cv.setMouseCallback`)

- OpenCV 도형 그리기 (`cv.circle`)

- 키보드 입력 처리 (`cv.waitKey`)

3️⃣ 마우스 영역 선택 및 ROI 추출

설명

사용자가 마우스로 이미지의 특정 영역을 선택하면 해당 영역을 ROI(Region of Interest) 로 추출하여 별도의 창에 표시하는 프로그램이다.
컴퓨터비전에서 자주 사용되는 관심 영역 추출 방법을 실습한다.

주요 기능

- 이미지 불러오기 및 화면 출력

- 마우스 드래그를 통해 사각형 영역 선택

- 선택한 영역을 ROI 이미지로 추출

- ROI를 별도의 창에 표시

- `r` 키 입력 시 선택 영역 초기화

- `s` 키 입력 시 ROI 이미지 파일 저장

사용 기술

- OpenCV 마우스 이벤트 (`cv.setMouseCallback`)

- OpenCV 사각형 그리기 (`cv.rectangle`)

- NumPy 슬라이싱을 이용한 ROI 추출

- 이미지 저장 (`cv.imwrite`)
